# Google_Map
A simple Python application that opens Google Maps directly from the terminal using either an address or latitude and longitude coordinates.

## Features
* Search locations using an address
* Search locations using latitude and longitude
* Automatically opens Google Maps in the default web browser
* Easy-to-use terminal interface

## Project Structure
```bash
google_mapit/
│
├── google_maps.py
├── README.md
└── requirements.txt
```

## Requirements
* Python 3.x
* Internet connection
* A web browser (Chrome, Edge, Firefox, etc.)

## Installation
1. Clone the repository:
git clone https://github.com/Keerthana-webdev/Google_Maps.git

2. Navigate to the project directory:
cd google_maps

3. No additional packages are required since `webbrowser` is a built-in Python module.

## Usage
```bash
### Run the program:
python maps.py

### Example 1: Search by Address
>> 1
Give the address >> Bengaluru

### Example 2: Search by Coordinates
>> 2
Latitude >> 12.9716
Longitude >> 77.5946
```
